# Категории и теги

Документ описывает подход к категориям и тегам в проекте **ВкусноТут**.

## Основная идея

Категории и теги решают разные задачи.

```txt
Категория = основной раздел сайта
Тег = дополнительный признак рецепта
```

Категория должна отвечать на вопрос:

```txt
Что это за тип блюда?
```

Тег должен отвечать на вопросы:

```txt
Как это готовится?
Чем это удобно?
Для какого случая подходит?
Какая особенность у рецепта?
```

## Почему не стоит копировать структуру книги

В исходных рецептах могут быть авторские разделы:

- минимум усилий;
- блюда в контейнерах;
- блюда в рукаве;
- фастфуд для похудения;
- во время переездов;
- новогодний стол.

Для книги такая структура нормальная, но для сайта удобнее использовать классическую навигацию.

Авторские разделы лучше превратить в теги или подборки.

## Планируемые категории

На первом этапе:

- Завтраки
- Супы
- Основные блюда
- Гарниры
- Перекусы
- Салаты
- Десерты
- Выпечка
- Соусы
- Напитки
- Праздничные блюда

## Правила категорий

1. У рецепта должна быть одна основная категория.
2. Категории не должны дублировать теги.
3. Категории должны быть понятны пользователю без контекста.
4. Категорий не должно быть слишком много.
5. Категории используются в основной навигации сайта.

## Планируемые теги

### Особенности

- быстро
- много белка
- низкокалорийное
- сытное
- простое
- для заготовки
- без готовки

### Способ приготовления

- запекаем
- жарим
- варим
- тушим
- блендерим

### Оборудование

- духовка
- сковорода
- аэрогриль
- микроволновка
- блендер
- мультиварка

### Приём пищи

- завтрак
- обед
- ужин
- перекус

### Повод

- праздничное
- на каждый день
- в дорогу
- на работу

## Типы тегов

В базе можно хранить тип тега.

Планируемые типы:

- `FEATURE`
- `METHOD`
- `EQUIPMENT`
- `MEAL`
- `DIET`
- `OCCASION`
- `SOURCE`

## Пример

Рецепт:

```txt
Паста с тунцом
```

Категория:

```txt
Основные блюда
```

Теги:

```txt
быстро
много белка
обед
ужин
без духовки
```

## Подборки

Подборки можно добавить позже.

Примеры будущих подборок:

- Быстрые завтраки
- Рецепты с творогом
- Блюда в духовке
- ПП-десерты
- Праздничные рецепты
- Рецепты без готовки

На первом этапе подборки можно не реализовывать.
